# Capital Quiz App

The **Capital Quiz App** is a fun and interactive quiz game built with React. Users can test their
knowledge of world capitals, track their score, and get instant feedback. Confetti animation is triggered
for scores above 8! 

## Features
1. Interactive multiple-choice questions.
2.Dynamic progress tracking with a progress bar.
3.Instant score display after completing the quiz.
4.Celebratory confetti for scores above 8.
5.Clean, responsive UI built with Tailwind CSS.

## Installation
1. Clone the repository: `git clone `
2. Navigate to the project folder: `cd capital-quiz`
3. Install dependencies: `npm install`
4. Start the development server: `npm start`
   
## Tech Stack
• React.js
• Tailwind CSS
• React Confetti
• JavaScript (ES6+)
